How does JavaScript work?

  • Sumaia
  • Web Design
  • 14

It Is a computer programming language also known as client-side scripting language which is run inside an internet browser. The client’s web browser needs to support the javascript it is the most crucial thing that should be remembered. The Working system of javascript is amazing. We can place some javascript code inside the normal webpage. The browser has a built-in interpreter when the browser loads the page, which is read the js code.

An execution context is created whenever a javascript program is run and also everything is happen inside the execution context. JavaScript is a Synchronous and single-threaded language.JavaScript code is executed in two phases: 1. Memory allocation phase: As a key-value pair, all the variables and functions of the javascript code get stored inside the memory component of the execution context. In the memory block, the whole function is been copied by the javascript code in a functional case. 2. Code Execution Phase: At a time javascript code is executed one line inside the code component of Execution Context.

Double equals is a comparison operator that is the necessary type of conversions before doing the equality comparison. It checks only value. Javascript converts any string to a number when we compare a string with a number. An empty string is converted to zero and a string that has no numeric value is converted to NaN. As a return we get false. On the otherhand, triple equal is also a comparison operator but it does compare strictly in javascript. It checks both values and types. If the two values are not the same after running the code it returns false. For example, if we would like to compare 2 with “2”, then it will return a false value.
